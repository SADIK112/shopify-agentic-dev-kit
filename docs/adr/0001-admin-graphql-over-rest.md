# ADR 0001 — Admin GraphQL over REST

**Status:** Accepted

## Context
Precise field selection lowers cost-based rate-limit usage; bulk operations and better pagination; REST is being de-emphasized by Shopify.

## Decision
REST allowed only where GraphQL cannot do the job; each exception gets its own ADR.

## Consequences
Recorded in `.ai/DECISIONS.md`. Revisit via a superseding ADR if conditions change.
